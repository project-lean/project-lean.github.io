What is "Project: LEAN"?

LEAN is about introducing developers to fast to develop and scalable development stacks. These stacks are specially helpful for startups and anyone developing a prototype/MVP that don't wan't to spend to release as fast as possible.

How does it work?

We are going to publish a series of guides about different technologies and by the end of each guide you will hopefully have fully functional prototype.

Can I get some help?

Sure thing! After all, helping is what this project is all about. You can talk to us about any of the published courses at our Gitter channel: project-lean.gitter.im

How can I help?

If you wan't to collaborate with a class about a technology stack, just open an issue with your suggested stack here: https://github.com/project-lean/project-lean.github.io/issues. If this stack isn't already covered we will create a repository for it on our organization, if the stack is already covered by a course you can always create pull requests to help us improve it
